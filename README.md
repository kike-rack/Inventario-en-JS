* Inventario para BroadSonick

** Descripcion

Este es un trabajo enmarcado en el bootcamp de Talento Digital de Chile. En el presente, se solicitó la entrega de un sistema de inventario para una tienda ficticia. En ese caso, se decide tematizar la tienda hacia el audio profesional.
Para cumplir con la pauta de evaluación, se decide utilizar un sistema a través de comandos de la consola. Si bien esto dota de dificultad en la usabilidad del sitio, se acompaña el proyecto con una interfaz informativa. La visualización se realizá utilizando Bootstrap, con colores sobrios y sobre todo simples. 

** Pendientes
